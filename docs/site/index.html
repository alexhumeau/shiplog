🚢 Shiplog

Push to main, get a changelog in Notion, Slack, and Markdown. Auto-generated from your git history.

brew install alexhumeau/tap/shiplog
🧠

AI-Powered Analysis

10 LLM providers: Claude, GPT, Gemini, Mistral, DeepSeek, Groq, Grok, Cohere, Together, Fireworks. Works without AI too.

📡

Multi-Destination

Write to Notion, Slack, Markdown, or JSON. Enable multiple writers at once — one push updates everything.

📸

Auto Screenshots

Convention-based screenshot capture. Map file patterns to routes, Shiplog captures them with headless Chrome.

How it works

git push → Shiplog triggers │ ├─ Collector: git log + diff + README + docs │ ├─ Analyzer: conventional commits + optional LLM │ ├─ Groups related commits │ ├─ Categorizes (feat/fix/refactor/docs/chore/perf) │ └─ Rewrites titles in plain language │ ├─ Screenshot: headless Chrome capture (optional) │ └─ Writers: multi-destination output ├─ Notion: rich database entries with dedup ├─ Slack: Block Kit webhook messages ├─ Markdown: CHANGELOG.md └─ JSON: structured stdout for piping

Quick Start

# Install brew install alexhumeau/tap/shiplog # Setup (creates .shiplog.yml + Notion DB) shiplog init # Preview shiplog run --dry-run --last 10 # Add to CI cp .github/workflows/shiplog.yml your-repo/
